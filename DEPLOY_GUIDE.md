# GrowTree 앱 배포 가이드

## 빠른 배포 방법

### 방법 1: Netlify Drop (가장 빠름) ⚡

1. https://app.netlify.com/drop 방문
2. `index.html`과 `app.js` 파일을 드래그 앤 드롭
3. 즉시 공유 가능한 링크 생성!

### 방법 2: GitHub Pages

1. GitHub에 새 저장소 생성
2. `index.html`과 `app.js` 파일 업로드
3. Settings > Pages에서 배포 활성화
4. `https://[사용자명].github.io/[저장소명]` 링크 생성

### 방법 3: Vercel

1. https://vercel.com 방문
2. 프로젝트 업로드
3. 자동으로 배포 링크 생성

## 배포 전 확인사항

- ✅ `index.html` 파일이 루트에 있음
- ✅ `app.js` 파일이 루트에 있음
- ✅ 모든 파일이 같은 폴더에 있음
